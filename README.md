# ecommerce-mern

An E-commerce Web App developed using MERN stack where user can buy and checkout product with ease and add their review and rating for the products.

## Features

- User Authentication
- Add Product to Cart
- Buy Product
- Payment Checkout
- Add Review
- Add Rating
